🌦️ Dynamic ETL Weather Dashboard 2.0

Dynamic ETL Weather Dashboard 2.0 is a modern weather analytics application that automates the Extract, Transform, and Load (ETL) process using real-time weather data from the OpenWeather API. The project provides interactive dashboards, historical weather storage, trend analysis, forecasting, and geospatial visualization through an intuitive Streamlit interface.

The application demonstrates real-world data engineering concepts by integrating API data collection, automated data transformation, database management, analytics, and interactive visualizations into a single end-to-end solution.

✨ Features
🌍 Real-time weather data for multiple cities
⚙️ Automated ETL pipeline
📊 Interactive analytics dashboard
📈 Weather trend visualization
🔮 Weather forecasting
🗺️ Interactive weather map
💾 SQLite database with SQLAlchemy ORM
🔄 Automatic data refresh
📦 Modular Python architecture
🧪 Unit tests with Pytest
🚀 CI/CD using GitHub Actions
